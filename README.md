# 💊 MediStock – Medical Inventory System

## 📌 Overview

MediStock is a smart medical inventory management system designed for small pharmacies. It helps manage stock, track expiry dates, and generate reports efficiently.

## 🚀 Features

* 📦 Batch-wise medicine tracking
* ⏳ Expiry date monitoring
* ➕ Add / update / delete stock
* 🧾 Purchase & sales records
* 📉 Write-off management
* 📄 PDF report generation

## 🌐 Live Demo
https://medistock-inventory-system.vercel.app

## 🛠️ Tech Stack

* HTML
* CSS
* JavaScript
* React (CDN)
* jsPDF
